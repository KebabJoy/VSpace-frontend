## Setup
Чтобы запустить проект, перейдите на [вебсайт](https://vtb-mu.vercel.app/),
или установите локально через npm.
```sh
git clone https://github.com/timursk/vtb.git
cd vtb
cd my-app
npm install
npm run dev
зайдите на http://localhost:3000/
```
